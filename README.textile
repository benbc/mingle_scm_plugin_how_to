h3. Overview

This is documentation for building a Mingle SCM integration. This documentation should be used in conjunction with the source code for the "Mingle Mercurial Plugin":http://github.com/ThoughtWorksStudios/mingle_hg_plugin

h3. Building the documentation

Requirements: 
* ruby and rake installed
* "Mingle Mercurial Plugin":http://github.com/ThoughtWorksStudios/mingle_hg_plugin repository cloned into a sibling directory (for purposes of adding code snippets)

The default rake task will output the documentation to a folder named 'build'
